# 温馨提示弹窗 💖

一个可爱的桌面提示工具，会在屏幕上随机位置弹出温馨提示语，提醒你关爱自己。

## ✨ 功能特点

- 🎨 **多彩渐变背景**：10种精美配色方案，每个弹窗随机选择
- 💬 **丰富提示语**：100+条温馨提示，关心你的生活点滴
- 😊 **可爱装饰**：20种表情符号随机装饰
- 🎯 **圆角设计**：现代化UI，带边框的圆角卡片样式
- ✨ **渐显动画**：弹窗出现时有柔和的淡入效果
- 🔝 **窗口置顶**：确保提示不会被其他窗口遮挡

## 📋 依赖环境

- Python 3.x
- tkinter（Python自带）
- Pillow（PIL）

## 🚀 安装依赖

```bash
pip install Pillow
```

## 💻 使用方法

直接运行程序：

```bash
python main.py
```

程序会持续在屏幕上随机位置弹出温馨提示窗口。

## ⚙️ 自定义配置

你可以在 `main.py` 中自定义以下内容：

### 修改提示语

编辑 `tips` 列表，添加或修改你喜欢的提示语：

```python
tips = [
    "要记得喝水哦(～￣▽￣)～",
    "你的自定义提示语",
    # ...
]
```

### 修改配色方案

编辑 `color_schemes` 列表，自定义背景色和边框色：

```python
color_schemes = [
    {"bg": "#FF6B9D", "border": "#C9184A"},  # 粉红渐变
    # 添加你的配色...
]
```

### 调整弹窗频率

修改 `start_spawning()` 函数中的时间间隔：

```python
time.sleep(random.uniform(0.5, 2.0))  # 0.5-2秒随机间隔
```

## 📝 注意事项

- 程序运行时会持续创建弹窗，建议在需要时运行
- 弹窗无法通过点击关闭，需要结束程序才能停止
- 适配 1920x1080 分辨率，其他分辨率可能需要调整坐标范围

## 🛑 停止程序

按 `Ctrl + C` 或直接关闭终端窗口即可停止程序。
